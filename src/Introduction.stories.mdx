# 欢迎使用 val-design

一个轻量级 React 组件库

## 📦 安装

```bash
npm i val-design
```

```bash
yarn add val-design
```

## 🔨 示例

```tsx
import { Button } from 'val-design'

const App = () => (
  <>
    <Button btnType="primary" size="large">
      val design
    </Button>
  </>
)
```

引入样式：

```tsx
import 'val-design/dist/index.css'
```
