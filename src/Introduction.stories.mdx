# 欢迎使用 val-design

一个轻量的 React 组件库

## 📦 安装

```bash
npm i val-design
```

```bash
yarn add val-design
```

## 🔨 示例

引入样式：

```jsx

import 'val-design/dist/index.css'

```

导入组件：

```jsx
import { Button } from 'val-design'

const App = () => (
  <>
    <Button type="primary" size="lg">
      val design
    </Button>
  </>
)
```
